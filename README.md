但是我比較在意的是易讀性，因此我挑選了一個架構非常清楚的模板，希望能讓將來看我履

額外功能:

	2.我在Personal Projects這個部分，使用了在html裡面再加一個html的方法，來把我之前在medium發表過的文章引用到裡面。

心得:

但我覺得我這個網站的可擴充性和易讀性都是極高的，將來想一直加入新的內容都是沒有問題。
同時他的框架分得非常明確，使用者一目了然，所以我才挑選這個模板，希望能在這個課程結束後，
